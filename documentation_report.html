Executive Summary

This document serves as the formal project documentation and UI/UX analysis for the updated Students Club Tharparkar (SCT) landing page. SCT is a registered non-profit student network whose core mission is to empower rural student leadership, bridge digital divides, and bring quality educational bootcamps directly to rural Sindh, Pakistan.

The landing page has been fully refactored to align with brand guidelines requiring a Navy Blue (`#1a237e`) and White scheme, backed by the elegant Poppins typeface. Using modern responsive layouts (Vite + React 19 + Tailwind CSS), this website serves as a high-fidelity portal to drive member onboarding, showcase events, collect donations, and coordinate educational initiatives.

STUDENTS CLUB THARPARKAR (SCT) 1. Project Introduction

1. Project Introduction

1.1 Purpose of the Website

The primary purpose of the SCT Landing Page is to serve as the unified digital hub for the organization. As a non-profit operating in a rural and digitally developing district, SCT requires a lightweight, highly responsive, and visually striking platform to share its activities, manage student cohorts, coordinate volunteer initiatives, and receive donor support. The site bridges the digital divide by displaying critical notifications, training programs, and community achievements.

1.2 Problem Statement

Students and youth in the Tharparkar region face significant barriers to quality digital skills training, guidance counseling, and career mentorship due to geographical isolation. Organizations like SCT work locally, but lacked a centralized, high-performance website that could:

  • Provide transparent data on their impact, programs, and outreach achievements.
  • Offer clean, accessible registration channels for bootcamps and workshops.
  • Present a professional interface to capture donations and regional partner sponsorships.
  • Maintain absolute performance on low-bandwidth networks common in rural villages.

1.3 Project Objectives

The development of this platform was directed by four central objectives:

  1. Brand Standardization: Align the visual language of SCT with its identity using a professional navy and white palette.
  2. High Onboarding Conversion: Provide a smooth, client-side validated contact and application portal for student programs.
  3. Interactive UX: Integrate lightweight, smooth animations that work natively across hardware without performance degradation.
  4. Accessibility (A11y): Enforce strict color contrast, semantic HTML, and accessibility attributes.

1.4 Target Audience

The target audience consists of students seeking counseling, digital training, and youth leadership initiatives, local and global donors wishing to support educational equity, regional partners (NGOs and IT hubs) sponsoring outreach drives, and active/prospective student volunteers.

STUDENTS CLUB THARPARKAR (SCT) 2. Project Scope

2. Project Scope

2.1 Included Features

The Landing Page integrates a comprehensive set of features, optimized for maximum frontend speed:

  • Responsive Navigation Bar: A sticky navigation bar supporting fluid scroll links, keyboard accessibility, and a mobile hamburger menu.
  • Dynamic Hero Section: Features custom particles and statistics cards integrated with custom animation frame counters.
  • CTA Donate Banner: Displays an active summer educational fund campaign, direct copy buttons for banking details, and quick contact anchors.
  • Structured About Us Section: Explains SCT’s mission, vision, and core pillars (Education, Leadership, Welfare).
  • Sponsor & Partners Grid: Displays logos of regional sponsors to build community trust.
  • Impact Numbers Counter: Renders interactive numerical counters for active members, villages covered, and student beneficiaries.
  • Find Yourself Benefit Grid: Details membership advantages with custom hover-glowing cards.
  • Photo Gallery: Showcase of workshops, clean drives, and literacy campaigns, supporting tag filters (All, Events, Workshops, Activities).
  • Upcoming Programs & Register: Shows active schedules, including capacities, locations, and direct registration triggers.
  • Meet Our Team: Team grid showcasing leadership profiles and social anchors.
  • Controlled Contact Form: Controlled inputs with field validation, dynamic error alerts, and accessible contrast drop-down options.

2.2 Benefits to Students Club Tharparkar

Enhanced Trust & Transparency: By listing actual statistics (Impact Numbers) and supporting partners, donors can verify the legitimacy of SCT drives.

Efficient Student Admissions: The upcoming programs listing includes locations and schedules with clear registration CTAs, allowing students to apply easily without having to visit the physical center.

Strong Regional Brand Placement: Moving to a unified Poppins typography and navy blue color scheme establishes SCT as a professional organization, attracting corporate sponsorship and university collaborations.

3. Technologies Used & Architecture

3.1 Technology Stack Selection

To ensure a fast, robust, and scalable platform, a modern frontend stack was chosen:

  • HTML5 & CSS3: Built using semantic structure and modern layout paradigms (CSS Custom Variables, Flexbox, and CSS Grids).
  • JavaScript (ES6+): Powers the application logic, custom events, and state mutations.
  • React 19 (Core Library): Utilizes React 19's virtual DOM, concurrent rendering engine, and hooks to handle state transitions.
  • Vite (Bundler & Tooling): Serves as the compilation engine, enabling hot module replacement (HMR) and optimized Rolldown assets bundling.
  • Tailwind CSS v3.4: A utility-first CSS framework used to build custom styles. Compiles into a single optimized payload.
  • Framer Motion v12: Drives advanced layout animations, scroll triggers, and tab swaps using hardware acceleration.

STUDENTS CLUB THARPARKAR (SCT) 5. UI/UX Design System

5. UI/UX Design System

5.1 Color Theme & Typography

The color palette is built around Navy Blue (`#1a237e`), representing trust and authority, and White, providing maximum readability. Golden accents highlight CTAs and active states:

Primary Navy (#1a237e): Main theme color.
Navy Mid (#283593): Card layers.
Accent Gold (#f5c518): CTAs & focus indicators.
White (#ffffff): Clean backgrounds and text contrast.

The typography uses the Poppins typeface. Using Poppins across headers and body copy creates a cohesive font style.

5.2 Layout & Responsiveness

The page layout uses dynamic Flexbox wraps and Tailwind CSS grid columns. Standard breakpoints are used:

xs (375px) Mobile sm (640px) Mobile Lg md (768px) Tablet lg (1024px) Laptop xl (1280px) Desktop

Cards dynamically re-order from single-column on mobile viewports to multi-column rows on desktops. All typography sizing uses clamped values to prevent layout breaks.

5.3 User Experience Features

  • Micro-animations: Subtle hover scales, card glows, and active tab transitions guide user focus.
  • A11y Compliance: All links contain `aria-label` tags, and the contact options use custom CSS colors to ensure contrast.
  • Interactive Modals: Clicking programs loads detailed modal overlays in the current viewport.

STUDENTS CLUB THARPARKAR (SCT) 9. Challenges Faced

9. Challenges Faced & Code Resolutions

9.1 React 19 CountUp Rendering Exception

Problem: Standard third-party countup libraries crashed when loading stats cards. React 19 concurrent features caused exceptions inside older library lifecycles, causing the site to display a blank white screen upon scroll.

Solution: Built a custom countup component using the native browser requestAnimationFrame API inside the codebase. It increments numbers using ease-out curves and handles clean component unmounting.

9.2 Viewport Intersection Hooks

Problem: Custom viewport listeners caused layout shifts on scroll.

Solution: Standardized animations using Framer Motion's native whileInView and viewport={{ once: true }} attributes, which leverage CSS transitions and hardware acceleration.

9.3 Windows OneDrive & Dev Environment Locks

Problem: The Vite compiler hung on `transforming...` during builds. OneDrive actively synced the Desktop workspace folder, holding locks on configurations and assets.

Solution: Created a PowerShell script ([local_build_npm.ps1]) to copy source files to a local temp folder outside OneDrive, run the build cleanly, and transfer the `dist` folder back.

9.4 Local Network git push Dropouts

Problem: Local network dropouts prevented pushing commits to GitHub.

Solution: Created a PowerShell REST script ([push_modifications.ps1]) that base64 encodes files and uploads them via the GitHub REST API contents endpoint.

11. Conclusion

The developed landing page for the Students Club Tharparkar (SCT) is a modern platform that combines aesthetic appeal with technical performance. Refactored into a Poppins and Navy Blue theme, the landing page establishes a professional brand voice for the organization.

From a technical standpoint, the project demonstrates senior-level frontend engineering. Implementing custom React 19 countups, viewport animations, and direct REST API deployment scripts resolved early runtime and environment issues. The resulting compiled bundles are highly optimized:
