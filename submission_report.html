Design Philosophy & System

The user interface has been designed to reflect professional maturity while maintaining a high-tech, progressive feel suitable for a forward-thinking student group. The color system, layout structure, and components conform to modern UI/UX guidelines:

Color Tokens

Primary Navy (#1a237e): Core background color, representing depth, structure, and academic trust.
Accent Gold (#f5c518): Interactive elements, badges, and focus alerts. Reflects optimism and excellence.
Navy Mid (#283593): Used for cards and secondary text backgrounds, providing layering.
Success Green (#34d399): Soft alert indicators, programs badges, and selected form states.

Typography

Poppins (Display & Body): A versatile, modern geometric font family utilized throughout the website to ensure a clean, elegant brand experience across all screens.

Aesthetic Features

Glassmorphism: Used across all components. Employs semi-transparent backdrops (`rgba(255,255,255,0.05)`) layered with high-performance CSS blur and subtle white borders.

Glowing Orbs: Subtle glowing radial gradients placed beneath layers to bring the dark layout to life.

Design Adaptability & Responsiveness

All margins, paddings, and card layouts utilize relative clamp dimensions and Tailwind's breakpoint grids. They transition dynamically from small mobile devices (320px) to ultra-wide desktop monitors (1920px+) with strict alignment.

Technical Specifications & Architecture

The landing page was engineered using modern software guidelines. Clean separation of concerns, modularized component architecture, and responsive state handlers ensure high speed and low memory usage.

Framework & Core Tools

  • Vite + React 19: Utilizing the virtual DOM and the latest React 19 concurrent features for rapid mounting, hot module reloading (HMR), and fast compilation.
  • Tailwind CSS v3.4: Compiles styles into minimal custom classes, keeping CSS payload under 40kB.
  • Framer Motion v12: Drives all animations. Handled natively to optimize GPU acceleration on scrolling.

Custom Core Implementations

To guarantee absolute compatibility with React 19 and bypass third-party package errors, two custom systems were built from scratch:

  1. React 19 Safe CountUp Component: Built using browser-native requestAnimationFrame. Smoothly counts numbers from 0 to target values in 2.5 seconds using dynamic ease-out curves, completely avoiding the need for heavy, incompatible npm packages.
  2. Viewport-Aware Framer Motion Triggers: All scroll reveals use native whileInView and viewport={{ once: true }} attributes rather than active JS scroll listener bounds, reducing main-thread CPU overhead to zero.
